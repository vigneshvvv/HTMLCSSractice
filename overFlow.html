The Etical hacking

Ethical hacking, also known as white hat hacking, is the practice of using hacking techniques to identify security vulnerabilities in systems and networks, but with the explicit permission of the owner. It's a proactive approach to cybersecurity aimed at strengthening defenses and preventing malicious attacks. Ethical hackers use the same tools and methods as malicious hackers, but with the goal of improving security rather than causing harm.The ethical hacing is all about creating back door in system to get terminal access and get access to your files. The ethical hacing practices include DDOS attack, SQL injection, File Upload vulnerability etc.
